Namespace: lcsCountries

lcsCountries

An object containing detailed information about countries around the world, including region codes, continent codes, capital cities, time zones, calling codes, and currency details. This object is intended for use in applications that require structured, country-specific information such as region, currency, and calling codes.
Properties:
Name Type Description
country_code Object Each property of `lcsCountries` represents a country by a unique identifier (lowercase country name with underscores). Each country's object contains detailed properties about that country.
Properties
Name Type Description
ip string Reserved for IP address or related data specific to the country.
name string The full name of the country (e.g., "Afghanistan").
region_code string The ISO 3166-1 alpha-2 region code for the country (e.g., "AF" for Afghanistan).
continent_code string The continent code where the country is located (e.g., "AS" for Asia).
country_capital string The capital city of the country (e.g., "Kabul").
timezone string The primary timezone of the country, typically the capital city's timezone (e.g., "Asia/Kabul").
calling_code string The country calling code used for international dialing (e.g., "93" for Afghanistan).
currency Object An object describing the country's currency details.
Properties
Name Type Description
label string The name of the country's currency (e.g., "afghani" for Afghanistan).
sign string The symbol of the currency used in the country (e.g., "؋" for Afghan afghani).

Example

// Accessing details of Afghanistan
const afghanistan = lcsCountries.afghanistan;
console.log(afghanistan.name); // "Afghanistan"
console.log(afghanistan.currency.sign); // "؋"
